Microservices:
Microservices are independent, self-contained services that communicate over a network (e.g., HTTP, gRPC).
Each microservice can be written in a different language or framework (e.g., Django, Node.js, Go).
Each microservice has its own database (or shares a database but manages its own schema).
Communication between microservices is done via APIs (e.g., REST, GraphQL).
